# CardWeaver

CardWeaver は、クリエイティブなアイデア創出のための「KJ法アプリケーション」です。

## コンセプト

「大切なアイデアをブラックボックス化せず、手元で安全に管理・拡張できるようにしたい」

多くのSaaSやサーバー管理型のホワイトボードアプリでは、ユーザーのアイデアがクラウドに依存することになり、セキュリティやデータの永続性に不安を感じることがあります。
CardWeaver はオープンソースソフトウェア (OSS) として、以下の価値を提供します。

- **ローカル完結**: データは JSON ファイルとしてローカルに保存され、完全なポータビリティを持ちます。
- **機密性の確保**: 外部サーバーへの送信を行わず、大切なアイデアを自前で管理できます。
- **拡張性**: C# (Windows Forms) で構築されており、独自の機能（プラグイン連携や新構造の導入）を追加しやすい設計を目指しています（随時アップデート予定）。

## 特徴

- カードとボックス（グループ化）による視覚的なアイデアの整理。
- ボックス内のネスト（ボックス・イン・ボックス機能）。
- OSのようなウインドウシステムに基づいた洗練されたZ優先順序（カードが最前面、ボックスが背面）。
- ライトモード / ダークモード対応。
- インフィニットサイズのキャンバスとズームイン・ズームアウト機能。

## ビルド環境

- OS: Windows OS
- 開発環境: Visual Studio 2022 または .NET CLI (C# Windows Forms Application)
- .NET のバージョン: .NET 6.0 以上

## 使い方

1. プロジェクトをクローンし、Visual Studio 等で開きます。
2. ビルドして実行すると、初期キャンバスが表示されます。
3. 左上のメニューにある「ファイル」->「名前を付けて保存」から、現在のワークステーション状態を JSON に保存できます。
4. アイデアのカードを追加し、ドラッグ＆ドロップで配置可能です。ボックスを追加して包含することで、論理的なグループ化を行うことができます。
